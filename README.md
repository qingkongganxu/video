# video

这是一个精简后的 Tampermonkey 音视频批量下载脚本仓库。

本仓库由以下两个仓库合并整理而来：

- `qingkongganxu/video-auto-download.git`
- `qingkongganxu/-.git`

当前已经移除 APK 分析资料、Node.js 爬虫源码、测试用例、依赖清单和本地依赖环境，只保留脚本使用、回退和查看修改记录所需的文件。

## 主脚本

推荐使用仓库根目录的 `asmrmoon-audio-downloader.user.js`，它和当前最新命名版本 `自动下载8.14.js` 内容一致。

直接把脚本导入或覆盖到 Tampermonkey 即可使用。

## 脚本能力

- ASMRMOON 目录/全站音视频扫描与批量下载。
- MP3、MP4、m3u8 自动识别并统一进入下载队列。
- m3u8 尽量合并为 MP4。
- 并发下载，默认 5，最高 20。
- 下载速度、总进度、失败重试、最终重试、暂停/恢复。
- 支持"暂停(非正在下载)"和"继续下载(非正在下载)"。
- 低内存目录保存模式，支持按分类和小目录写入本地文件夹。
- 通用页面媒体扫描，只处理当前页面已经明文暴露且当前会话可访问的媒体链接。
- **分类文件夹复用**：检测到本地已存在相似名称的分类文件夹时，直接使用该文件夹，避免创建多个相似文件夹。

脚本不绕过登录、付费墙、DRM、反盗链、网站限速或其它访问控制。

## 版本规则

- 格式：大版本号.小版本号（如 8.14，8 是大版本号，14 是小版本号）
- 若下次修改未超过 24 小时：小版本号 +1，大版本号不变
- 若隔了 24 小时：大版本号 +1，小版本号清零
